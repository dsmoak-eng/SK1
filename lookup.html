DART Corp Code Lookup

Find 8-digit corp_codes by scanning recent DART filings for each company
The DART API needs an 8-digit corp_code (e.g. 00126380 for Samsung), not the 6-digit stock code (005930).
This tool finds corp_codes by searching recent filings for each stock code — no large ZIP download needed.

Note: Companies that haven't filed anything in the last 3 months may not be found.
