DART Corp Code Lookup

Find 8-digit corp_codes for your watchlist. These are needed by the DART API.
The DART API uses an 8-digit corp_code (고유번호) per company, NOT the 6-digit stock code.
Example: Samsung = stock code 005930, corp_code 00126380.
This tool downloads the official mapping from DART and extracts your companies.
